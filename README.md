1) Variables - 

Struct and classes - Made classes for boss, and player character each classes have HP, attack damage, and name. You can initiate those variables under int main() by using setMaxHealth(), setAttDamage and setName(), , but it also has default variables. You can see those values by doing getMaxHealth(), getAttDamage(), and getName().
